# Blaze Writer AI - Landing Page

This is a professional frontend development project that I completed as a skill evaluation task. The assignment required building a modern, responsive landing page with advanced component architecture, TypeScript implementation, and custom CSS styling. Through this project, I demonstrated my proficiency in React ecosystem, responsive design, and clean code practices using industry-standard technologies like Next.js, TypeScript, and CSS Modules.
